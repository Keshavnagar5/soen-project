A scalable real-time chat application built using modern full-stack technologies. The platform supports real-time messaging, AI-powered smart replies, secure authentication, and high concurrency with low latency.

Tech Stack:
Frontend: React.js, HTML5, CSS3, JavaScript, WebSocket Client |
Backend: Node.js, Express.js, WebSockets (Socket.IO), JWT Authentication |
Database & Caching: MongoDB, Redis |
AI Integration: Google Gemini AI |
Tools: Git, GitHub, Postman (API Testing)
